# Fourier Optics Note-2 文章设计

## 目标

将 `src/content/blog/FourierOptics2.md` 写成一篇可直接发布的标量衍射工程笔记。文章面向已经了解复振幅与二维傅里叶变换、但不希望追踪完整数学推导的读者，重点回答三个问题：每种理论描述了怎样的物理图景、引入了哪些近似、工程计算时应当选择哪一种方法。

## 内容边界

- 讨论单色相干光在均匀介质中的标量传播。
- 保留能体现传播机制和近似条件的关键公式，不展开 Green 定理或衍射积分的完整推导。
- 解释 Huygens–Fresnel、Kirchhoff、Rayleigh–Sommerfeld、Fresnel 积分与角谱法之间的关系。
- 简要指出标量理论的失效场景，但不展开 Debye、FDTD、FEM 或 RCWA。
- 工程选型使用普通段落和项目符号，不使用表格。

## 组织方式

文章采用三条线汇合的结构，而不把所有理论误写成简单的前后替代关系：

1. 直觉线：从 Huygens–Fresnel 的次级波源图景进入 Fresnel 积分。
2. 边值问题线：从 Helmholtz 方程的边界重建思想进入 Kirchhoff 与 Rayleigh–Sommerfeld 解。
3. 计算线：用平面波分解解释角谱理论，并说明它与空间域传播积分的等价关系。

最后用模型关系总结、工程选型建议和数值检查清单将三条线汇合。

## 章节设计

1. **问题与共同前提**：定义“已知一个平面的复振幅，求另一位置的光场”，声明单色、相干、均匀介质和标量近似。
2. **Huygens–Fresnel 原理**：用次级球面波的相干叠加建立直觉，解释距离、相位、球面扩散和倾斜因子。
3. **Fresnel 衍射积分**：引入传播距离的二次相位近似，给出常用积分与卷积形式，强调其本质是近轴近似。
4. **Kirchhoff 衍射理论**：将衍射描述为 Helmholtz 边值问题，解释经典孔径边界假设及其数学不一致性。
5. **Rayleigh–Sommerfeld 解**：说明一致边界条件带来的改进，重点介绍第一类积分及标量理论内部的适用范围。
6. **角谱理论**：将输入场分解为不同方向的平面波，解释传播波、倏逝波、传播传递函数和 FFT 实现风险。
7. **模型关系**：区分物理原理、边值积分、近轴近似和频域表示，澄清 Fresnel 与角谱法的位置。
8. **工程选型**：用项目符号说明不同问题适合的方法，不使用表格。
9. **计算前检查清单**：检查标量近似、最大传播角、采样、混叠、倏逝波和结果验证。

## 写作与验收标准

- 中文为主，首次出现的重要术语附英文名称。
- 每个核心模型至少包含物理图景、关键公式、使用条件和局限性。
- 不把 Fresnel number 当作 Fresnel 近似成立的唯一判据。
- 明确角谱法与 Rayleigh–Sommerfeld 是同一标量传播问题的不同表示。
- 修正 frontmatter 中的 `Schalar`，并统一使用 `Kirchhoff` 与 `Rayleigh–Sommerfeld`。
- Markdown frontmatter、数学块和标题层级能够通过站点构建。
